第一个小难题 Navigator

基本组件直接写 react 组件就好了,但都是单独个体,抛去组件样式类型的不完善外,就是没有 react-router 那么方便的路由工具
可能是因为更新了最新的 0.43 原因,部分 navigator 组件都出现各种报错
最后选择了

“react-navigation”: “git+https://github.com/react-community/react-navigation.git#7edd9a7"
