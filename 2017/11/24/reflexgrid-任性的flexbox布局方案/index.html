reflexgrid - 任性的flexbox布局方案

Posted on 2017-11-24   |   In Tools

Flexbox

前端在现代浏览器中布局的利器,但是… 

兼容性 - 前端开发的噩梦啊,至少对我来说,兼容是肯定郁闷的,生命在流逝…
IE8/IE9…… 绝对是怨念啊

我在项目中引入了 Reflex grid, 来保住我们心爱的 flexbox

Reflex grid 能干嘛啊

  • 轻量级 压缩+gzip 约 2kb
  • 用法遵循flexbox,上手成本低
  • 跨浏览器支持
  • 可支持一定定制化,基于SASS/SCSS

详情可见官网
官网地址

使用体验:

  • 好好测试, 毕竟这种东西不可能做到完美兼容各种浏览器. 特别情况还是要特殊对待
  • 对于习惯工作在现代浏览器上的朋友,工作效率大大提升
